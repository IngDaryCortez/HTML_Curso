Formato de texto HTML

Los elemento de formato se diseñaron para mostrar tipo especiales de texto

  • < b > - Texto en negrita
  • < strong > - Texto Importante
  • < i > - Texto Cursiva
  • < em > - Texto Enfatizado
  • < mark > - Texto Marcado
  • < small > - Texto mas pequeño
  • < del > - Texto Eliminado
  • < ins > - Texto Insertado
  • < sub > - Texto Subíndice

  • < sup > - Texto SupeÍndice
  • < code > - Define una pieza de código de computadora

Elementos de citas y citas HTML

HTML < blockquote> para sitas

Los elemento HTML < blockquote > define una sección que cita de otra fuente.

Los navegadores suelen hacer sangría en los elementos < blockquote > .

Here is a quote from WWF's website:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

HTML < q> para citas cortas

La < q > etiqueta HTML define una cita corta

Los navegadores normalmente insertan comillas alrededor de la cita.

WWF's goal is to: Build a future where people live in harmony with nature.
